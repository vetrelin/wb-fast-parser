# WB-Fast-Parser | Асинхронный парсер Wildberries

Высокопроизводительный инструмент на Python для автоматического сбора данных с маркетплейса Wildberries. Предназначен для мониторинга цен и анализа конкурентов.

---

### 🛠 Стек технологий:
* **Python 3.10+**
* **Playwright** — имитация поведения пользователя для обхода 403 ошибок.
* **Pandas / Openpyxl** — качественная обработка данных и экспорт в Excel.

### ⚡️ Основные возможности:
- **Обход защиты:** использование браузерного движка Chromium для работы с динамическим контентом.
- **Интеллектуальный экспорт:** автоматическое формирование Excel-отчетов (Бренд, Название, Цена).
- **Headless mode:** возможность работы в фоновом режиме без открытия окна браузера.

### 📦 Запуск:
1. Установить зависимости: `pip install playwright pandas openpyxl`
2. Установить браузер: `playwright install chromium`
3. Запустить: `python parser.py`
